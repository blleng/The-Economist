# The Economist Weekly Archive

Welcome to my personal archive of *The Economist*. 

The archives are regularly updated on Every **Sunday 01:00 (UTC+8)**.

## Retention Policy

To keep the repository clean and avoid unnecessary storage bloat, this repository enforces a **rolling 12-issue retention policy**, ensuring only the 12 most recent weeks are kept.

## Directory Structure

Files are organized into folders categorized by the publication date:
```
TE-YYYY-MM-DD/
├── TE-YYYY-MM-DD.epub
├── TE-YYYY-MM-DD.mobi
├── TE-YYYY-MM-DD.azw3
└── TE-YYYY-MM-DD.pdf
```
